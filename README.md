# Do Healthy (두헬시)

건강한 라이프스타일을 위한 맞춤형 레시피 추천 서비스

## 📌 프로젝트 소개

Do Healthy는 개인의 건강 상태와 목표에 맞춘 레시피를 추천해주는 맞춤형 서비스입니다.  
단순한 레시피 제공을 넘어서, 사용자의 건강 데이터를 기반으로 개인화된 식단을 제안합니다.

## 🚀 시작하기

```bash
# 저장소 클론
git clone https://github.com/[username]/do-healthy.git

# 의존성 설치
pnpm install

# 개발 서버 실행
pnpm run dev

# 스토리북 실행
pnpm run storybook
```
